## 36NEXT 是什么
36NEXT 是一个 [NEXT](http://next.36kr.com/) 的非官方 iOS 客户端。

NEXT 是一个快速发现、分享和讨论新产品的社区。在这里，你可以第一时间发现国内外最新、最酷、最好玩的互联网产品，也可分享自己最喜爱的产品，参与圈内人士的讨论。[36Kr](http://www.36kr.com/) 出品。

上 NEXT，每天刷新更酷的生活方式。
